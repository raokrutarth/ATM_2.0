Detailed Description

Represents a minutia.

This minutia representation assumes that the angle is represented in radians.

The hash code of these objects uses four bytes per minutia: Minutia.X in the left most 11 bits; Minutia.Y in the next 11 bits; Minutia.Angle in the next eight bits; and Minutia.MinutiaType in the last two bits. Therefore, this method is ineffective for values of Minutia.X or Minutia.Y greater than 2047.
