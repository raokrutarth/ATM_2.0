Detailed Description

An implementation of the OrientationImage extractor proposed by Sherlock et al. in 1994.

This is an implementation of the OrientationImage extractor proposed by Sherlock et al. [1] in 1994.

We apply a 3x3 median filter to improve the computed OrientationImage.

References:

  1. B. G. Sherlock, D. M. Monro, and K. Millard, "Fingerprint enhancement by directional Fourier filtering," IEE Proceedings Vision Image and Signal Processing, vol. 141, no. 2, pp. 87-94, 1994.
